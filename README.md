# mybolg
项目地址:[www.dengzl.cn]
#### 项目介绍 
是本人第一个Java练手项目。
#### 项目当前功能
- 用户登录
- 后台文章管理
- 文章展示
- 标签查找
- 分类查找
- 文章编辑
- 浏览管理
#### 项目当前环境

|  工具 | 名称 
| ------------ | ------------
| 开发工具  | IDEA 
|  语言 | JDK1.8、HTML、css、js 
| 数据库  | Mysql
| 项目框架  | SSM 
| ORM  | Mybatis 
| 安全框架  | SpringSecurity 
| 项目构建  | Maven 
| 运行环境  | 腾讯云windows server 2008R2
| 对象存储  | 腾讯云cos
